# keycodes

Javascript DOM event key codes map.

## Install

```sh
npm install @afractal/keycodes
yarn add @afractal/keycodes
```

## Usage

```typescript
import keycodes from "keycodes";
const enterCode = keycodes["enter"]; // returns 13
```
